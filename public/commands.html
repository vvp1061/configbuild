Вернуться к конфигуратору

Загрузка файлов на сервер

Выберите один или несколько файлов для загрузки в папку uploads/.

Выполнение простых команд

Выберите команды, которые будут выполнены на сервере.

Выполнение скрипта my_script.sh

Выберите опцию, которая будет передана в скрипт в качестве аргумента.


Создание архива с результатами

Нажмите кнопку, чтобы запустить обрабатывающий скрипт. После его выполнения будет создан ZIP-архив с результатами, который сразу же начнется скачиваться.
